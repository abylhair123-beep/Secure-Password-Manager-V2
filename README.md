
# Менеджер паролей V2

Программа для хранения паролей с безопасным шифрованием. Есть тёмная тема, генератор паролей, импорт/экспорт и поиск по сервису/логину.  

---

## Что умеет

- Тёмная тема для удобства  
- Хранение паролей в зашифрованном виде  
- Добавление, удаление и просмотр паролей  
- Генератор случайных паролей  
- Копирование пароля в буфер обмена  
- Поиск по сервису или логину  
- Экспорт и импорт паролей в CSV  
- Русский интерфейс  

---

## Как пользоваться

1. При первом запуске создаёшь мастер-пароль.  
2. Входишь в приложение с ним.  
3. Добавляешь пароли для сайтов и сервисов.  
4. Можно сгенерировать сильный пароль и сразу добавить его.  
5. Можно экспортировать или импортировать пароли в CSV.  
6. Для поиска используешь поле поиска сверху.
